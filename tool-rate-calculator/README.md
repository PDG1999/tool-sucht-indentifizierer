# 💰 Rate-Calculator - SAMEBI Tools

## 🎯 Übersicht

**Stundensatz-Rechner für Psychologen**

Hilft Psychologen bei der optimalen Preisgestaltung durch Berechnung von:

- Gewünschtes Netto-Einkommen
